# APP頁面設計 - Stock KOL Tracker

## 📱 頁面架構總覽

基於"瞎忙散戶"Persona的設計原則：
- **極簡化操作**：最多3步驟完成記錄
- **視覺化呈現**：一眼看出重要資訊
- **情緒化管理**：幫助用戶冷靜決策

---

## 🏠 主要頁面設計

### 1. 首頁 (Home) - 個人投資筆記
**設計理念**：個人化的投資追蹤中心，專注於KOL發言與投資標的的關聯

#### 頁面元素
```
┌─────────────────────────────────────┐
│  📊 追蹤概覽 (點擊進入個人投資日誌)   │
│  ┌─────────────────────────────────┐ │
│  │ AAPL  📈 +2.5%  [線圖縮圖]      │ │
│  │ TSLA  📉 -1.2%  [線圖縮圖]      │ │
│  │ MSFT  📈 +0.8%  [線圖縮圖]      │ │
│  │ NVDA  📈 +3.1%  [線圖縮圖]      │ │
│  └─────────────────────────────────┘ │
│                                     │
│  ┌─────────────┐ ┌─────────────┐    │
│  │  📝 快速紀錄 │ │  👤 KOL雷達  │    │
│  └─────────────┘ └─────────────┘    │
└─────────────────────────────────────┘
```

#### 功能重點
- **追蹤概覽**：顯示追蹤中的股票標的、昨日漲跌、線圖縮圖
- **快速紀錄**：一鍵進入記錄KOL發言流程
- **KOL雷達**：一鍵進入KOL分析頁面

## 🎨 設計原則

### 1. 極簡化操作
- **一鍵記錄**：最多3步驟完成
- **自動識別**：減少手動輸入
- **快速查看**：重要資訊一目了然

### 2. 視覺化呈現
- **色彩編碼**：綠色=看多，紅色=看空，灰色=中性
- **圖表直觀**：用圖表代替文字
- **進度條**：顯示準確率和進度

### 3. 情緒化管理
- **正面鼓勵**：正確決策時給予鼓勵
- **溫和提醒**：錯誤決策時溫和提醒
- **風險警告**：及時提醒風險
